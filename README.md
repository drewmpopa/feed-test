# Not a game of perfect

<img src="https://media.shortform.com/covers/png/golf-is-not-a-game-of-perfect-cover@8x.png" width="250">

[Golf Is Not A Game of Perfect](https://www.goodreads.com/book/show/935617.Golf_is_Not_a_Game_of_Perfect) is a book by Bob Rotella. I have been listening to [the audiobook](https://www.youtube.com/watch?v=-L97cjemM5k&t=993s) daily since first reading it back in 2023.

The goal of this page is to document personal revelations, provide practical tips for fellow golfers seeking improvement, as well as searching for universal truths that athletes and professionals of all kinds can learn from.
